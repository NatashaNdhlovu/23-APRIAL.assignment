# Simple Calculator (Python)

## 📌 Project Description

This project is a simple command-line calculator written in Python. It allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division. Each operation is implemented using a separate function, and the program includes error handling for division by zero.

---

## ✨ Features

* Add two numbers
* Subtract two numbers
* Multiply two numbers
* Divide two numbers
* Handles division by zero errors
* Takes user input from the command line

---

## 🧮 Supported Operations

| Symbol | Operation      |
| ------ | -------------- |
| `+`    | Addition       |
| `-`    | Subtraction    |
| `*`    | Multiplication |
| `/`    | Division       |

---

## 🛠️ How the Program Works

1. The user is prompted to enter two numbers.
2. The user selects an arithmetic operation (`+`, `-`, `*`, `/`).
3. The program calls the corresponding function.
4. The result is displayed on the screen.
5. If division by zero is attempted, an error message is shown.

---

## ▶️ How to Run the Program

1. Make sure you have **Python 3** installed.
2. Save the file as `calculator.py`.
3. Open a terminal or command prompt.
4. Run the program using:

   ```bash
   python calculator.py
   ```

---

## 📄 Example Usage

```
Enter your first number: 10
Enter your second number: 5
Choose an option (+, -, *, /): /
Result: 2.0
```

Division by zero example:

```
Enter your first number: 10
Enter your second number: 0
Choose an option (+, -, *, /): /
Result: Error: Cannot divide by zero
```
